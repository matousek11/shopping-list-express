# Shopping list
API built on Express.js with MongoDB

# Installation

1. `yarn install`
2. Set up `.env` based on `.env.example`
3. `yarn dev`

## For production

1. `yarn install`
2. Set up `.env` based on `.env.example`
3. `yarn build`
4. `yarn start`

# Testing

- jest - for unit tests
- supertest - for api tests
- `yarn test`

# API documentation

- on `localhost:3000/api-docs/`
